= 画面仕様書[アカウント作成]

== 概要
新規でアカウントを作成する画面。
登録後は２パターンの画面に遷移する。

* 招待なしに作成した場合 → [プロジェクト設定画面]に遷移する。メンバー招待はこの時点ではできないか、登録後に画面を別途開く。
* メールの招待を受けて作成した場合 → 招待を受けたプロジェクトのメイン画面に遷移する
